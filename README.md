# Portfolio Website

A responsive, single-page portfolio website showcasing my experience, projects, and technical skills. Built with vanilla HTML, CSS, and JavaScript.

## Features

- **Dark Theme Design** - Clean, modern dark grey color scheme
- **Responsive Layout** - Mobile-first design that works on all devices
- **Smooth Scrolling** - Navigation with smooth scroll animations
- **Interactive Elements** - Hover effects and scroll-triggered animations
- **Contact Integration** - Direct links to email, LinkedIn, and GitHub

## Sections

- **Home/Introduction** - Personal introduction with location and resume link
- **Tech Skills** - Categorized technical skills with icons
- **Experience** - Professional experience and projects
- **Projects** - Showcase of portfolio projects
- **Contact** - Contact information and social links

## Technologies Used

- HTML5
- CSS3 (with CSS Custom Properties)
- Vanilla JavaScript
- Intersection Observer API (for scroll animations)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/dmien47/portfolio.git
   ```

2. Open `index.html` in your web browser

That's it! No build process or dependencies required.

## Customization

To customize this portfolio for your own use:

1. Replace placeholder content in `index.html`:
   - Update personal information (name, title, location)
   - Add your experience entries
   - Add your projects
   - Update contact links

2. Modify colors in `styles.css`:
   - Update CSS custom properties in `:root` to change the color scheme

3. Add your resume:
   - Place your resume PDF in the `assets` folder
   - Update the resume link in `index.html`
